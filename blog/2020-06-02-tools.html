Tools and services used to develop WOWidget

2nd of June 2020

Programming

  • Xcode
  • App Store
  • GitHub
  • Swift, Storyboard

I've chosen Storyboard over SwiftUI because I wanted to support iOS 12. According to the App Store statistics it is still used on ~20% devices.

Design

  • Photopea

Photopea is a free „Photoshop” in a browser. Supporting PSD, Sketch and many other formats.

Video

  • Mousecape
  • QuickTime Player
  • Final Cut Pro X Trial

I wrote about recording videos.

Website

  • Namecheap
  • GitHub
  • Netlify
  • Yandex.Mail
  • Sublime Text

Namecheap is the only paid service apart from the App Store subscription.
Website content is stored on GitHub and every commit is automatically deployed by Netlify to their servers and CDN.
Yandex provides free email hosting with a custom domain without advertisements. Zoho used to be an option, but they stopped supporting IMAP on their free plan.
